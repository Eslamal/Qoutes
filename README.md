## Overview
A beautifully designed Quotes App built with Android technologies, providing users with inspirational quotes, bookmarking features, and the ability to share quotes with friends. 
The app is designed for simplicity and usability, leveraging modern development practices.

## Features
- Random Quote Generator: Fetch a random inspirational quote with a swipe.
- Bookmark Quotes: Save your favorite quotes for easy access later.
- Share Quotes: Share your favorite quotes directly with friends or on social media.
- Swipe Interaction: Use intuitive swipe gestures to explore new quotes.
- Offline Support: Bookmark and view quotes even without an internet connection.

## Technologies
- Kotlin: Programming language for Android development.
- Material Design
- LifeCycle Components
- Room Database 
- Coroutines
- Retrofit 
- Navigation Components
